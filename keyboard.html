Installation Instructions

MacOS

  1. Download the keyboard layout file below
  2. Copy the file to ~/Library/Keyboard Layouts/
  3. Log out and log back in
  4. Go to System Preferences → Keyboard → Keyboard Layouts
  5. Click the + button and find your custom layout
  6. Select it and enable "Show Input menu in menu bar"
